# Rapport — TP1

## Points clés

### 1. Variables CSS 
Les variables CSS sont définies dans `:root` et vont centraliser une bonne partie des couleurs/espacements
qui serton réutilisées plus tard

### 2. Mise en page avec Flexbox et Grid
Le layout utilise sur Flexbox pour le header, le hero et le footer,
et sur CSS Grid avec `auto-fit / minmax` pour la grille de projets
Pour les espacements, il n'y a pas eu de choix particulier, j'ai séléctionné la variable qui me semblait 
la plus adaptée à chauqe fois 
### 3. Responsive design avec media query
Le responsive s'est surtout concentré sur la compression d'éléments
au milieu de l'écran
